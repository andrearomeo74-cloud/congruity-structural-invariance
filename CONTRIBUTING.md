# Contributing

Thank you for your interest in this repository.

This project is primarily intended as a scientific reproducibility companion for the Congruity structural invariance preprint.

## Scope of contributions

Constructive contributions are welcome, especially regarding:

- reproducibility improvements
- benchmark validation
- bug fixes
- documentation improvements
- dataset compatibility improvements
- robustness testing
- methodological critique

## Out of scope

This repository is not intended for:

- proprietary Congruity engine development
- commercial implementation support
- advanced orchestration system contributions
- unpublished framework extensions

## Reporting issues

If you encounter:

- broken scripts
- dependency conflicts
- benchmark inconsistencies
- reproducibility failures

please open a GitHub issue with:

- operating system
- Python version
- executed script
- error traceback
- reproduction steps

## Coding expectations

Please:

- keep code readable
- prefer explicit implementations
- preserve reproducibility
- document assumptions clearly

## Scientific integrity

Negative results are welcome.

This repository explicitly values falsification, robustness testing, and transparent methodological critique.
