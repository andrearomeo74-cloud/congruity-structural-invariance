# Data

This repository uses publicly available benchmark datasets.

Datasets are not redistributed unless explicitly permitted by their original licenses.

## Included data sources

### Breast Cancer

Wisconsin Breast Cancer dataset

Available through:

```python
sklearn.datasets.load_breast_cancer()
```

---

### Diabetes

Diabetes benchmark dataset

Available through:

```python
sklearn.datasets.load_diabetes()
```

---

### EEG Motor Imagery

PhysioNet EEG Motor Movement/Imagery Dataset

Source:
https://physionet.org/content/eegmmidb/

Access via:

```python
mne.datasets.eegbci
```

---

### Biological Aging

Biological aging benchmark data used in this study are derived from public biomarker/methylation-compatible benchmark sources.

Specific preprocessing pipelines are documented in the notebooks.

---

### Synthetic Ecology

Synthetic stochastic Lotka–Volterra simulations are generated programmatically within this repository.

No external download required.
